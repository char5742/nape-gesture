# Sources/nape-gesture-core-tests/AGENTS.md

この階層は `NapeGestureCore` の実行可能テストを担当します。外部テストフレームワークへ依存せず、CI とローカルで同じコマンドを使って検証できる状態を保ちます。

## テスト方針

- 新しい core 不変条件を追加したら、この executable test に失敗時の具体的な診断を追加する。
- 正常系だけでなく、通ってはいけない設定、境界値、復帰条件、欠落 fixture 相当の退行を固定する。
- テストは実機、TCC、現在時刻、ホームディレクトリ、ローカル絶対パスへ依存させない。
- 失敗時の出力は日本語で、どの不変条件が壊れたか分かる内容にする。
